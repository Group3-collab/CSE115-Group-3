# Hangman Game in C Language

## Project Description
This is a very simple game called "Hangman Game" for our CSE115 course's group project. 
Players will guess letters to reveal a hidden random word until they run out of tries.

## Project Structure
- `main.c` - Main game controller
- `requirements.txt` - Development requirements
- `support/` - Supporting Game components
- `data/` - Word database
- `others/` - Project documentation
